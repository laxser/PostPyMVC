PostPyMVC 是一个类似于Ruby上的Sinatra的超轻量级支持MVC的Web框架。严格意义上来说这类框架不能称为框架，因为他们将服务器，处理逻辑，业务逻辑，代码都写在一起。但是，要是你熟悉或者使用过Ruby上的Sinatra你就会懂，这个框架用来写Web应用，尤其是急功近利的Web应用是多么的方便。

PostPyMVC本身基于Python的WSGI，模板方面使用Python的Genshi。这两个都是比较非主流小众的模块。因此你需要使用以下的python模块才能使用PPM的功能。至于HelloWorld么，您应该已经可以自行研究了～
